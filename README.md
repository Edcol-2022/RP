这里机场有一个敌对NPC载具刷新点 利用这里我们刷经验 移动鼠标镜头对向车头 选择载具导弹 按住右键 这样可以刷新被摧毁的敌对载具
注意经验值 如果摧毁敌对车辆经验不加了 按G 刚刚的粘弹炸毁布里凯德 任务结束 选择重新开始 就这样循环 任务时间超过15分钟 效果更佳
